Automatic Translations

DESCRIPTION

Generate translations for transcripts your videos into many languages automatically, using the power of AWS Cloud Computing technologies (Amazon Web Services) you can enjoy in your favorite language and with the control of each time section the content of the video . A perfect way to avoid missing relevant data from your multimedia content.

FEATURES
  • Translations: Translate the Video Transcriptions faster and automatically to many languages, just one click.
  • Video tracker: Manage the video through its text tracks (subtitles) and take the play control which is playing or not
  • Presentation: Change the presentation of the Transcription for you reading mode easier. Subtitles or Paragraphs
  • Manage subtitles translated for each one video
  • Full responsive.
  • Support for Google Chrome, Firefox, Microsoft Edge.

KarlChat v1.0

DESCRIPTION

It's a Simple Chat App using HTML, CSS, ReactJS, Redux, Node, Firebase, Google Authentication. KarlChat is an instant messaging application for smartphones and browsers, in which messages are sent and received through the Internet, this version has authentication implemented thanks to Firebase Authentication, which works by logging in with Gmail accounts. Among its functions are to create groups or chat rooms and to review the messages that have been sent. This version is still very basic.

FEATURES
  • Send and receive messages.
  • Create a new Chat Room.
  • Message storage in Firebase Cloud Firestore.
  • Animations.
  • Full responsive.
  • Authenticate with Firebase.
  • Dynamic avatar for each Group.
  • Two Themes(Light and Dark).
  • Supported in Google Chrome, Firefox, Microsoft Edge browsers.
